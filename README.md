A full-stack **Jira-inspired project management application** built using Next.js App Router.  
It enables teams to manage projects using **Kanban boards**, lists, and cards with a modern UI and smooth drag-and-drop interactions.

---

##  Tech Stack

- **Frontend & Backend:** Next.js (App Router)
- **Database:** MongoDB, Mongoose
- **Authentication:** NextAuth.js (Credentials-based, RBAC-ready)
- **Drag & Drop:** @dnd-kit
- **Styling:** Vanilla CSS (Glassmorphism UI, Dark Mode)
- **Font:** Geist (via `next/font`)

##  Features

- **Full-Stack Architecture** using Next.js App Router and MongoDB
- **Authentication System** with NextAuth.js (RBAC-ready)
- **Dashboard** to create and manage project boards
- **Kanban Board View** with Lists and Cards
- **Drag & Drop Support**
  - Move cards between lists
  - Reorder lists dynamically
- **Premium UI**
  - Glassmorphism design
  - Dark mode
  - Fully responsive layout
- **Optimized APIs** for efficient list and card reordering
